Git is essential for any modern developer. Version control, collaboration, and cloud backup. 

# Initialize Git repository
git init

# Add files to staging
git add .

# Create commit with message
git commit -m "Initial commit: HTML Pokedex structure"

# Connect to GitHub
git remote add origin https://github.com/usuario/html-pokedex.git

# Push changes
git push -u origin main
