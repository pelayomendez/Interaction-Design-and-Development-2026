Every modern web application starts with semantic HTML. Let's look at the basic structure of our Pokédex project. 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pokédex</title>
    <link rel="stylesheet" href="./css/main.css">
</head>
<body>
    <header>
        <nav class="nav">
            <img src="./img/logo.png" alt="Logo Pokédex">
            <ul class="nav-list">
                <li><button id="ver-todos">Ver todos</button></li>
            </ul>
        </nav>
    </header>
    <main>
        <div id="listaPokemon"></div>
    </main>
    <script src="./js/main.js"></script>
</body>
</html>
