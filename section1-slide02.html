WHAT IS ASTRO?

MODERN AND FAST
FRAMEWORK

Astro is a web framework that ships zero JavaScript by default, loading code only when necessary. Perfect for fast content sites.

FAST

Zero JS by default, loads only what's needed

FLEXIBLE

Use React, Vue, Svelte or vanilla

SIMPLE

Familiar syntax, easy to learn 

# Create new Astro project
npm create astro@latest

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
