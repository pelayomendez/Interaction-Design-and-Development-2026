A database is a structured system for storing, querying, and updating data. Without one, every page refresh loses everything.

WITHOUT A DATABASE

let todos = []  // resets on refresh
todos.push({ text: "buy milk" })
// gone when you close the tab

WITH A DATABASE

await supabase
  .from('todos')
  .insert({ text: 'buy milk' })
// persisted forever, accessible
// from any device, any session
