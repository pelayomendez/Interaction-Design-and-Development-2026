FULL STACK + AI
WORKFLOW

PLAN → SPEC
→ BUILD → REVIEW

1. PLAN — Define goal, user, constraints
2. SPEC — Write acceptance criteria in plain language
3. BUILD — Hand the spec to Claude/AI
4. REVIEW — Compare output to spec, not to vibes
5. ITERATE — Update spec, not just the prompt

A good spec takes 10 minutes. It saves 2 hours of back-and-forth.
