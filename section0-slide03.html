MODERN CSS DESIGN

CSS3 allows us to create modern and responsive interfaces. We use CSS variables, Grid Layout, and Flexbox to structure our Pokédex. 

:root {
    --clr-black: #1c1c1c;
    --clr-gray: #ececec;
    --clr-white: #f7f7f7;
    
    --type-fire: #F08030;
    --type-water: #6890F0;
    --type-grass: #78C850;
}

.pokemon-todos {
    display: grid;
    grid-template-columns: 1fr;
    gap: 1rem;
}

@media screen and (min-width: 700px) {
    .pokemon-todos {
        grid-template-columns: 1fr 1fr 1fr;
    }
}
