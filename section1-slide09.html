PASSING DATA
BETWEEN COMPONENTS

Components receive data through props, like in React. TypeScript helps us with typing. 

---
// components/PokemonCard.astro
interface Props {
    name: string;
    url: string;
    types?: string[];
}

const { name, url, types = [] } = Astro.props;
---

<div class="card">
    <h3>{name}</h3>
    {types.map(type => <span class={type}>{type}</span>)}
</div>
