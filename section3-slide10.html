Diagram of the data flow in the complete full-stack application: from the user's click to the image generated by Gemini.

1. User → Click on "Generate Pokemon"
2. Frontend (Astro/React) → fetch() a Flask API
3. Backend (Flask) → Receives POST request
4. Gemini API → Generates Pokemon image
5. Flask → Encodes base64 and responds with JSON
6. Frontend → Renders image in <img>

Total time: ~2-4 seconds (depends on Gemini API)
