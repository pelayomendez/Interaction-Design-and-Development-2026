COMPONENT

Astro components have two parts: the code (frontmatter) and the HTML template. Simple and powerful. 

---
// Component Script (Frontmatter)
// Executes on the server at build time
interface Props {
    name: string;
    url: string;
}

const { name, url } = Astro.props;

const response = await fetch(url);
const data = await response.json();
---

<!-- Component Template -->
<div class="pokemon-card">
    <img src={data.sprites.front_default} alt={name} />
    <h3>{name.toUpperCase()}</h3>
</div>

<style>
    .pokemon-card {
        border: 2px solid #ccc;
        border-radius: 8px;
        padding: 1rem;
    }
</style>

The code between --- executes only on the server
