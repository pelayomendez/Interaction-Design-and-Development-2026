Layouts allow sharing common structure between pages. Define once, use everywhere. 

---
// layouts/Layout.astro
import Header from '../components/Header.astro';

interface Props {
    title: string;
}

const { title } = Astro.props;
---

<!doctype html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <title>{title}</title>
    </head>
    <body>
        <Header />
        <main>
            <slot />  <!-- Page content -->
        </main>
    </body>
</html>

The <slot /> is replaced with the content of each page
