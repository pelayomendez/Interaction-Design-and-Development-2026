ROUTING

Astro automatically creates routes based on the file structure in the pages/ folder. 

src/pages/
├── index.astro            →  /
├── pokedex.astro          →  /pokedex
├── about.astro            →  /about
└── pokemon/
    └── [id].astro         →  /pokemon/:id (dynamic)


// Use the Layout in a page
---
import Layout from '../layouts/Layout.astro';
---

<Layout title="My Pokédex">
    <h1>Welcome to the Pokédex!</h1>
    <PokemonGallery />
</Layout>

index.astro, about.astro

DYNAMIC
