CORS CONFIGURATION

CORS (Cross-Origin Resource Sharing) allows your frontend (port 4321) to access the backend (port 5000). Flask requires flask-cors to be configured. 

from flask import Flask
from flask_cors import CORS

app = Flask(__name__)

# Enable CORS for all origins (development)
CORS(app)

# Or specific for production
CORS(app, resources={
    r"/api/*": {
        "origins": ["https://mi-frontend.com"],
        "methods": ["GET", "POST"],
        "allow_headers": ["Content-Type"]
    }
})

@app.route('/api/data')
def get_data():
    return jsonify({'message': 'CORS enabled!'})

Security: In production, limit specific origins
