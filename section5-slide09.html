Supabase Realtime uses PostgreSQL's built-in change notifications to push updates to all connected clients instantly — no polling, no websocket server to maintain.

SUBSCRIBE TO CHANGES

const channel = supabase
  .channel('todos-changes')
  .on(
    'postgres_changes',
    {
      event: '*',      // INSERT, UPDATE, DELETE
      schema: 'public',
      table: 'todos'
    },
    (payload) => {
      console.log('Change!', payload)
      // payload.new — new row
      // payload.old — old row
      // payload.eventType — what happened
    }
  )
  .subscribe()
