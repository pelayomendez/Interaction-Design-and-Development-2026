QUERY LANGUAGE

SQL is the language you use to talk to relational databases. Four operations cover 90% of what you'll ever need.

READ & FILTER

-- Get all users
SELECT * FROM users;

-- Filter by condition
SELECT name, email
FROM users
WHERE active = true;

-- Sort results
SELECT * FROM posts
ORDER BY created_at DESC
LIMIT 10;

WRITE & DELETE

-- Create a record
INSERT INTO users (name, email)
VALUES ('Ana', 'ana@test.com');

-- Update a record
UPDATE users
SET active = false
WHERE id = 42;

-- Delete a record
DELETE FROM users
WHERE id = 42;
