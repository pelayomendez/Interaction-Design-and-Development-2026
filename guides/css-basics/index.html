Basics 1

What is CSS?

Cascading Style Sheets — the language of visual design on the web

CSS controls how HTML elements look in the browser.

CSS lets you control:

  • Colors, fonts, and text styling
  • Spacing, sizing, and layout
  • Animations and transitions
  • Responsive design for different screen sizes

Why it matters:

  • Separates content (HTML) from presentation (CSS)
  • One stylesheet can style an entire website
  • Change the look without touching the HTML
  • Without CSS, browsers apply basic default styles
style.css 
/* CSS makes HTML look good */
body {
  font-family: sans-serif;
  background: #f8f9fc;
  color: #111827;
}

h1 {
  font-size: 2.5rem;
  font-weight: 800;
  color: #6366f1;
}

p {
  line-height: 1.6;
  max-width: 65ch;
}

a {
  color: #6366f1;
  text-decoration: underline;
}
Basics 2

CSS Syntax — Rules, Selectors, Properties

Every CSS style follows the same pattern

A CSS rule has a selector and a declaration block.

selector { property: value; }
Targets elements Declaration

The parts:

  • Selector — targets which HTML elements to style
  • Property — what aspect to change (color, size, spacing...)
  • Value — the new setting for that property
  • Declaration — a property + value pair, ends with ;
💡

A rule can have multiple declarations inside the curly braces — one per line.

rule.css 
/* Element selector — targets all <h1> tags */
h1 {
  color: blue;
  font-size: 2rem;
}

/* Class selector — targets class="card" */
.card {
  background: white;
  padding: 1.5rem;
  border-radius: 8px;
}

/* ID selector — targets id="header" */
#header {
  background: navy;
  color: white;
}

/* Multiple selectors at once */
h1, h2, h3 {
  font-weight: 700;
  line-height: 1.2;
}
Basics 3

Three Ways to Add CSS

External, internal, or inline — choose wisely

There are three ways to connect CSS to your HTML.

1
External stylesheet (recommended)

Separate .css file linked with <link>. Best for real projects.

2
Internal stylesheet

<style> tag in the HTML <head>. OK for small demos.

3
Inline styles (avoid)

style="" attribute on an element. Hard to maintain.

💡

Always prefer external stylesheets — they keep your HTML clean and can be reused across pages.

external.html 
<!-- 1. External — best practice -->
<head>
  <link rel="stylesheet"
        href="styles.css">
</head>
internal-and-inline.html 
<!-- 2. Internal — in the <head> -->
<style>
  h1 {
    color: blue;
  }
</style>

<!-- 3. Inline — avoid this -->
<h1 style="color: blue">
  Hello
</h1>
