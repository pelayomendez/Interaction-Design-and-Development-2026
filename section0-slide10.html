GitHub Pages allows publishing static websites directly from a repository. Perfect for HTML/CSS/JS projects like our Pokédex.

DEPLOYMENT STEPS:

1. Push code to GitHub
2. Go to Settings → Pages
3. Select branch (main)
4. Select folder (/ root)
5. Save and wait for deployment
6. Site live in minutes!
